# dotfiles

## Getting Started (Linux)

```shell
curl -L https://raw.githubusercontent.com/yutoma/dotfiles/master/setup | sh
```

or

```shell
git clone --recursive --depth=1 https://github.com/yutoma/dotfiles.git ~/dotfiles
~/dotfiles/apply
```
